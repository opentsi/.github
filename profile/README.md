## Open Time Series Initiative 👋

Kicked off at useR! 2024 in Salzburg, Austria the project works to make [KOF Swiss Economic Institute](https://kof.ethz.ch)'s proven data sourcing pipelines publicly available and reusable. We are working to make data and the processing framework available for researchers. In the future, the [opentsi GitHub organization](https://github.com/opentsi) will host regularly updated time series data archives. The first public release is scheduled for the *[Swiss Statistics Meeting (Schweizer Statistiktage)](https://stat.ch/en/swiss_statistics_meeting)* at the end of August 2026.
